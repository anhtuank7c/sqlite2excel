# Sqlite to Excel

# Setup

Clone this project to your computer and install required dependencies

```bash
# Windows
python -m venv .venv
.\.venv\Scripts\activate
pip install -r .\requirements.txt

# Mac
python3 -m venv venv
source ./venv/bin/activate
pip install -r ./requirements.txt
```

# Run project

```bash
# Windows
python .\main.py

# Mac
python ./main.py
```
